2. Project ideas

This will be the project that support the users in the area of health. There are many people have high/low blood pressure, and many stroke death caused by this issue. This project will created an app that link the smartphone with a smart watch or any kind of sphygmomanometer and it will alert when the blood pressure meet the emergency state. Besides, it will help the user create a lifestyle that can help them in their blood pressure issue. There are many people is the victim of high/low blood pressure. In Vietnam, there are ¼ of adult has high/low blood pressure. However, to recognize how high or low their blood pressure is, they have to use the sphygmomanometer to see it. It is not that they can check it all the time and when the blood pressure is actually high or low to the alert point, they cannot know. This is the main reason lead to the stroke. Also, it is the motivation for the project to create an app that link the phone with a sphygmomanometer so when the user’ blood pressure reach the alert point, it will have an alarm to let the user know before it too late. This will be an app that launch in mobile device like smartphone, tablet, and smart watch. The app will have the common sphygmomanometer program. It will show the blood pressure rate of the user. This function help it act like a mobile sphygmomanometer which there are apps that have the same function already. Moreover, the app will have another function that collect the data of the user and create a suitable life schedule and meal for the users since the some type of food and activities might bring terribly bad affect to their health. Also it let the user keep on track of their blood pressure to show how improve they have been doing. First, to use the app, the user need to have a brief blood pressure test so the app can know the user blood pressure. Then, the user need to add their personal information like age, daily activities and their current health state, etc. Then, they will have the choice to sign up or not. This account will let the user save their information so when they want to change the device, all the information and the process will be easily transfer. The next step will be the user add the phone number of their family, relatives or people who close to them. With this feature, when the user’s blood pressure at the alert state, the app will alarm and if the user does not turn off the alarm, it will alert the other in case the user was faint or unable to call for help. This feature will help reduce the unfortunate risk from people that live on their own or any kind of scenario that they faint when no one surrounded. There are many programming language that can help to finish the app like JavaScript, C++, Python, Go, etc. and whichever language can help me do the project. However, I would like to use C++ or Python since I have studied a bit C++ when I was in high school and intro to IT course also provide some Python knowledge. In this course I have also learned HTML programming language but this language is not suitable for making app instead it focuses more in making website so maybe I could use HTML language to make a homepage website for the app. To create the app, the first skill needed would be IT skills which are the skills to program the app and let the app function smoothly. Besides the IT skill, it also need the medical knowledge since this is a medical app. Before make the app, we need to work with people who have the profession in the medical area. We need to validate the information from the doctor to get the accurate material to make the app and align the type of possible patient so that there will be no mistake when dialog the app user’ health state. When the project success, it would attract the interest of high and low blood pressure people. With the uses of the apps, it will minimize the damage from the disease and help the users have a better life in a convenient way. There are many people do not know how to actually deal with their blood pressure. The app can provide the knowledge to fill the knowledge the user lack of so that the users can get a healthier life. Block chain and cryptocurrencies

Blockchain is the thing store a set of data piece which are blocks and in each block have multiple transaction and it will be all change when there is a transaction happen whereas cryptocurrencies is the digital currencies that can be trade on the internet. Its value is determined by the user. The blockchain can be called as the tool to help us find the cryptocurrencies. There are many misunderstand about cryptocurrencies in some areas and Vietnam is one of these areas. People usually say that cryptocurrencies is a scam. However, it is a fact the cryptocurrencies is relatively valuable. Take Bitcoin, the most common and valuable cryptocurrencies in the current market. Bitcoin has the price of slightly more than 500 million vnd per coin and overall it is increasing day by day. Bitcoin has this value because several reasons. First, it is rare, there are only total of 21 million Bitcoin in the world. Yet back then, about 10 years ago, the value of Bitcoin is nothing because people do not consider its value. However, it becomes more expensive now because it started to have its value on the consumer and the rate of inflation. There are only 21 million Bitcoin so it literately cannot generate more no matter what so its value will increase consistently. Second, it has the fastest way to transfer money from a place to the other. For example, if you want to transfer the money to someone in a common way, it would be easy if you sit next to a person and give it to them. However, if you want to make a transaction from Ho Chi Minh City to Hanoi, it will take days and if you want to make a transaction from Vietnam to the U.S. it takes weeks. On the other hand, with cryptocurrencies like Bitcoin, you only need a few second to transact to anywhere worldwide. In addition, blockchain is help the cryptocurrencies system become invincible, in this case I mean cannot be hacked. In theory, it can be if the hacker have the amount of device equal to the amount of the Bitcoin device and hack in the same time which is not possible. That is why the Bitcoin can be totally trusted. Overall, cryptocurrencies is the digital money whereas blockchain is a system that record all the transaction in all the device the running cryptocurrencies. Cryptocurrency has been changed constantly since it was invented, not the way but the value of it. Its value in short-term might increase or decrease but in long-term it always increase. Therefore, there is a high possibility that the cryptocurrency will increase in the next 3 years and it might become the future money in the future. Moreover, the system of cryptocurrencies is relatively save with blockchain in the current period so I have no idea if it will have any improvement in the future. Nowadays, cryptocurrencies is not yet a common currency on the world. However, it has a huge potential but it is not that everyone can realize the potential of it. In the future, we might use cryptocurrencies instead of the currencies that we are using right now. With all the benefit that we can get from the cryptocurrencies that I mentioned above, cryptocurrencies is capability to be the future currencies. In consequence to this scenario, the numerous of bank would cease because there no longer needed to store the money. In addition, the current paper money might be disappear. All the transaction would become the transactions of cryptocurrencies. There are many case that banks got hack and lost a significant amount of money.
